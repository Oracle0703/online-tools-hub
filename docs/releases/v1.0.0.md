# Online Tools Hub v1.0.0

v1.0 把十二个隐私优先的浏览器工具升级为一个可组合、可取消、可离线、可验证隐私边界的本地工作流操作系统。

## 主要变化

- 发布六个策划工作流，覆盖 Base64、JSON、YAML、CSV、URL、JWT、PNG 与 SHA-256 任务；
- 提供移动端优先的 Workflow Studio、逐步预览、硬取消、清空和受限文件批处理；
- 建立构建期白名单 Operation、独占 Worker、Payload Vault、内存预算与失败恢复协议；
- 配方导入导出、运行摘要和隐私回执不包含正文、文件名或内容哈希；
- PWA 改为最小应用壳与用户主动下载的完整离线包；
- 隐私与能力中心提供机器可读清单和用户主动触发的合成自检；
- 首页、全站搜索、指南、sitemap、canonical、HowTo 与 Breadcrumb 已接入工作流内容。

## 隐私与安全边界

- 用户正文、文件和中间结果只存在当前标签页内存，不上传、不写入 URL、不持久化；
- 工作流只允许当前构建内的白名单 Operation，不执行任意脚本或远程模块；
- 取消、清空、页面离开、Worker 超时与崩溃都会释放运行资源；
- Base64、哈希与 JWT 解码均不等同于加密、身份验证或签名验证；
- 本项目不提供账号、云同步、远程 URL 抓取或输入内容分享链接。

## 发布验证

候选提交通过：

- Prettier、ESLint、Astro 类型检查与 1,122 项单元测试；
- Chromium、Firefox、WebKit 端到端测试；
- 真实 Microsoft Edge / Windows 2025 与 Safari / macOS 15；
- 移动端 Lighthouse 的 Performance、Accessibility、Best Practices 与 SEO 门禁；
- 360px、键盘、axe、CSP、PWA、隐私 canary、Operation chunk 与完整页面资源图预算。

正式标签固定指向合并提交 `0a611e42d0617ff586b9d8cdae2c60454eac6f2e`。详细证据映射见 [`docs/V1_RELEASE_CHECKLIST.md`](https://github.com/Oracle0703/online-tools-hub/blob/main/docs/V1_RELEASE_CHECKLIST.md)。

## 使用与升级

- 在线使用：<https://oracle0703.github.io/online-tools-hub/>
- 工作流目录：<https://oracle0703.github.io/online-tools-hub/workflows/>
- 隐私与能力中心：<https://oracle0703.github.io/online-tools-hub/privacy/>

这是静态站点版本，不需要账号或数据迁移。已安装的 PWA 会按版本化资源清单提示更新；完整离线包仍由用户主动下载或删除。

## 已知限制

- v1.0 只提供线性工作流，不支持循环、条件分支、任意节点图和用户脚本；
- 自定义工作流必须先从内置模板进入，真正的空白 Studio 与本地配方库属于 v1.1；
- 完整英文内容尚未发布，站点不会暴露半翻译的 `/en/` 路由。
